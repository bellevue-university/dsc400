## Week 1

title: Introduction to Big Data

### Description

Description for the first week

### Objectives

- Create a basic end-to-end analytics workflow
- Describe the differences between descriptive, diagnostic, predictive, and prescriptive analytics
- Estimate the key metrics of big data sources including storage and bandwidth requirements
- Understand how to big data applies to a specific industry or domain

### Readings

- Chapter 1 in **Big Data Science and Analytics**

### Supplemental Readings

### Resources

### Exercise

title: Evaluating big data stacks

### Discussion

title: Big data applications and use cases

#### Instructions

Research a domain specific application of big data. You can pick an example from the book or a domain that is of personal interest. Discuss how big data is being used or could be used in that domain. Describe the key data sources and analysis being used. Use at least two online resources from reputable sources.
